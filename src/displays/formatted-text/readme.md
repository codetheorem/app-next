# Formatted text

Renders the value with font styling based on it's options.

## Props
| Prop                | Description | Default        | Type      |
|---------------------|-------------|----------------|-----------|
| `value`             |             | `null`         | `String`  |
| `interface-options` |             | `null`         | `Object`  |
| `bold`              |             | `false`        | `Boolean` |
| `subdued`           |             | `false`        | `Boolean` |
| `font`              |             | `'sans-serif'` | `String`  |
