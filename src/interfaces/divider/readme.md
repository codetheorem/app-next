# Divider

## Props
| Prop    | Description                    | Default | Type     |
|---------|--------------------------------|---------|----------|
| `color` | Divider and title color        | `null`  | `String` |
| `icon`  | Icon rendered before the title | `null`  | `String` |
| `title` | Title for divider              | `null`  | `String` |
