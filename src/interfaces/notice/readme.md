# Notice

## Props
| Prop    | Description                   | Default                   | Type     |
|---------|-------------------------------|---------------------------|----------|
| `color` | Notice color                  | `'normal'`                | `String` |
| `icon`  | Icon rendered before the text | `'info'`                  | `String` |
| `text`  | Text for notice               | `'No text configured...'` | `String` |
