# Slider

## Props
| Prop            | Description              | Default | Type      |
|-----------------|--------------------------|---------|-----------|
| `disabled`      | Disabled                 | `false` | `Boolean` |
| `min-value`     | Minimum value            | `null`  | `Number`  |
| `max-value`     | Maximum value            | `null`  | `Number`  |
| `step-interval` | Interval of number steps | `1`     | `Number`  |
| `value`         |                          | `null`  | `Number`  |
